# Personal Portfolio Website

A responsive full-stack personal portfolio website developed using PHP, MySQL, JavaScript, AJAX, HTML, and CSS.

## Features

- Responsive modern UI
- Dark mode support
- Typing animation
- Scroll animations
- Admin login system
- Session & cookie authentication
- Password hashing
- AJAX project loading
- Contact form
- CV download button
- Education & experience sections
- Social media integration
- MySQL database integration

## Technologies Used

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- AJAX

## Admin Panel

The project includes an admin dashboard where projects can be:

- Added
- Edited
- Deleted
